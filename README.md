## Prerequisites

Following packages are required before installing the service:

* python3
* pip3

## Installation

````
git clone git@github.com:Simple-codinger/EEPROMProgrammer.git
cd EEPROMProgrammer
./dev-tools/install.sh
````

## Run

To start the script, run the following command:
````
./dev-tools/run.sh
````
